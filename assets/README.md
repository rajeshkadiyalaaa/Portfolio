# Source assets

Place original PNG/SVG files here before running `npm run optimize:assets`. Outputs are written to `public/` as WebP.

| Source | Output |
|--------|--------|
| `mountain-sun.png` | `public/mountain-sun.webp` |
| `about-footer.png` | `public/about-footer.webp`, `about-footer-1280.webp` |
| `projects-footer.png` | `public/projects-footer.webp`, `projects-footer-1280.webp` |
| `mountain-art.svg` | `public/mountain-art.webp` |
| `mountain-right.png` | `public/mountain-right.webp` |

Project PDF previews live in `public/project-previews/` (copies may be kept in `assets/source/previews/` for editing).
