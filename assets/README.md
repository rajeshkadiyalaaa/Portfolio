# Source assets

High-resolution originals for `npm run optimize:assets`. Outputs are written to `public/` as WebP.

| Source | Output |
|--------|--------|
| `about-footer.png` | `public/about-footer.webp`, `about-footer-1280.webp` |
| `projects-footer.png` | `public/projects-footer.webp`, `projects-footer-1280.webp` |
| `mountain-art.svg` | `public/mountain-art.webp` |
| `mountain-right.png` | `public/mountain-right.webp` |

**Project PDFs** — edit in `assets/source/previews/`, then copy to `public/project-previews/` using kebab-case names (`ai-assistant.pdf`, etc.) and update `src/data/projects.ts`.

**Section PNG art** — ink-wash corners live only in `public/background/` (not regenerated by the script).
